Adicionando WebHook

To set up a webhook, go to the settings page of your repository or organization. From there, click Webhooks, then Add webhook.

  1. Em Payload URL coloque o seguinte: 
    https://lucasheber-notifyme.herokuapp.com/
  2. Em Content type: 
    x-www-form-urlencoded
  3. Em Which events would you like to trigger this webhook? selecione 
    Just the push event
  4. Clique em 
    ADD WebHook
